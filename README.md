# RedolenceDB
A community-driven fragrance catalogue and collection tracker. Browse perfumes, rate and review them, track your collection, get personalised recommendations, and discover layering combinations.

---

## Features

### Catalogue

Browse and search perfumes by name, brand, concentration, and notes
Detailed perfume pages with fragrance notes, community ratings, sillage/longevity/gender/season consensus, and reviews

### Brand pages
Discover new, amazing brands and their fragrances

### Collection & Wishlist

Add perfumes to your personal collection with purchase date, bottle size, price, and personal notes
Track collection value and stats
Wishlist for perfumes you want

### Ratings & Reviews

Rate perfumes 1–5 stars
Log sillage, longevity, gender lean, and best season
Write reviews visible to the community

### Layering Recommendations

Community-voted perfume pairings with descriptions
Upvote pairings you agree with (one vote per user, persisted across sessions)
Suggest new pairings directly from the rating form

### Personalised Recommendations

8-question scent quiz covering mood, note preferences, occasion, season, sillage, longevity, and dislikes
Recommendations scored by quiz profile and collection signal (what you already own)
Negative tag penalty for notes you dislike
Match percentage displayed per perfume

### User Profile

View your recent ratings with reviews
Scent profile tags from the quiz
Stats: collection size, wishlist count, ratings given, average score, favourite season

### Admin Dashboard

Manage perfumes (add, edit, delete) with notes, tags, concentration, release and discontinuation dates
Manage brands and fragrance notes (with scent family, photo, description)
Moderate reviews — remove review text or delete ratings entirely
Manage layering pairs — edit descriptions, adjust vote counts, delete pairs
Site-wide stats: perfumes, brands, users, ratings

### Query console

Query tables in the database directly from the platform using the query console.
